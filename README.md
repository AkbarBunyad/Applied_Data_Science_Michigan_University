# Applied_Data_Science_Michigan_University

This repository contains solutions to the 5 courses under the Specialization - Applied Data Science with Python by University of Michigan on Coursera alongside some notes taken if needed:

1. [Introduction to Data Science in Python](https://www.coursera.org/learn/python-data-analysis)
2. [Applied Plotting, Charting & Data Representation in Python](https://www.coursera.org/learn/python-plotting/home/welcome)
3. [Applied Machine Learning in Python](https://www.coursera.org/learn/python-machine-learning/home/welcome)
4. [Applied Text Mining in Python](https://www.coursera.org/learn/python-text-mining/home/welcome)
5. [Applied Social Network Analysis in Python](https://www.coursera.org/learn/python-social-network-analysis/home/welcome)
